#  FastAPI & MongoDB To-Do List API
A robust, scalable backend service for managing tasks.

## Features
* **FastAPI:** High-performance, async-ready web framework.
* **MongoDB:** NoSQL database used for flexible data storage.
* **Full CRUD Implementation:** Supports all core API operations.

##  Technology Stack
* **Python**
* **FastAPI**
* **PyMongo**
* **MongoDB**
